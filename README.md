This is a Bluemap Addon that renders the Create Contraptions in bluemap. It will also generate the train models for https://github.com/BeneHenke/CreateTrainWebAPI. To install it simply put it into the Bluemap packs folder. To get more of the Create blocks rendered use it with https://github.com/Uiniel/BlueMapModelLoaders and https://github.com/Uiniel/BlueMap-Create-Resource-Pack.
